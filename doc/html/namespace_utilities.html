◆ ReadLists()

std::unordered_map< std::string, std::vector< std::string > > Utilities::ReadLists ( const std::string &  filename)

Reads data from a text file into a hashmap. The format of the file contents is expected to be:
Heading \n
Comma-separated list \n
(Empty line) \n
The file isn't a valid CSV file, as the lines are expected to have different numbers of elements. But each line is formatted as a valid CSV record (i.e., comma-separated and using quotes for special characters, etc).

Parameters
&filenameA string, the name of the text file
Returns
a hashmap {heading => data, ...}.
Exceptions
std::runtime_errorif filename can't be opened
std::runtime_errorif file contents aren't as expected
