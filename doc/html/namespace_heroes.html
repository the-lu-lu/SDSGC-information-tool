void Heroes::MakeHeroes ( )

This function processes all the data files. Calls ReadHeroesData, which actually constructs the heroes. It should be the only way heroes are created and it should be called exactly once.

Exceptions
std::runtime_errorif the function is called while heroes is not empty.
